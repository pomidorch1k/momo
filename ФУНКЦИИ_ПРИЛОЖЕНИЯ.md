# Функции приложения (по макету и коду)

Ниже перечислены экраны и функции, которые реализованы в веб-приложении и должны быть в мобильном приложении для Android (то же самое после сборки через Capacitor).

---

## 1. Главная (каталог товаров)

- Список товаров карточками.
- В каждой карточке:
  - изображение товара;
  - название;
  - описание (кратко);
  - цена (с учётом скидки);
  - старая цена зачёркнута при скидке;
  - бейдж скидки в процентах (например, «-15%»);
  - наличие: «В наличии: N шт» или «Нет в наличии»;
  - кнопка «Добавить в корзину» (только для авторизованных; для гостя скрыта или недоступна).

---

## 2. Корзина

- Доступ только для авторизованных пользователей. Гость видит экран «Войдите в аккаунт» с кнопкой «Войти».
- Список товаров в корзине:
  - картинка, название, цена со скидкой, количество;
  - кнопки «−» и «+» для изменения количества;
  - кнопка удаления товара из корзины.
- Кнопка «Очистить» корзину.
- Блок «Итого» с суммой и кнопкой «Оформить заказ».
- Если корзина пуста — сообщение и кнопка «Перейти к покупкам».

---

## 3. Вход и регистрация

- Переключение: «Вход» / «Регистрация».
- Поля:
  - при регистрации: имя, email, пароль;
  - при входе: email, пароль.
- Кнопки: «Войти» / «Зарегистрироваться».
- Ссылка «Продолжить как гость».
- Подсказка про назначение роли админа (через отдельный процесс/страницу).

---

## 4. Нижняя навигация (Navbar)

- **Главная** — каталог.
- **Корзина** — корзина (с бейджем количества товаров).
- **Админ** — только для пользователей с ролью `admin`.
- **Вход** или **Выход** — в зависимости от авторизации.

---

## 5. Админ-панель (только для роли admin)

- Список товаров с кнопками «Редактировать» и «Удалить».
- Кнопка «Добавить товар».
- Модальное окно добавления/редактирования товара:
  - название;
  - цена (₽);
  - скидка (%);
  - количество на складе;
  - URL изображения;
  - описание.
- Подтверждение при удалении товара.

---

## 6. Роли и доступ

- **Гость** — только просмотр каталога, корзина недоступна.
- **Пользователь (user)** — каталог + корзина + вход/выход.
- **Админ (admin)** — всё то же + админ-панель (CRUD товаров).

---

## 7. Бэкенд и данные

- Авторизация: **Supabase Auth** (вход/регистрация).
- Товары, корзина, роли: ваши **Edge Functions** (Supabase) — те же API, что использует веб-приложение.

При сборке Android через Capacitor все эти функции остаются без изменений: приложение открывает собранный веб-интерфейс в WebView и подключается к тем же API.
