  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Мобильное приложение для Android

  Проект настроен для сборки в Android-приложение через **Capacitor**. Список функций приложения (по макету и коду) описан в файле **ФУНКЦИИ_ПРИЛОЖЕНИЯ.md**. Инструкция по сборке — в **ANDROID.md** (нужны Node.js 20+ и Android Studio).
